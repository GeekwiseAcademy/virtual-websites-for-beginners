HTML Syntax

Tag Usage Example

<element>CONTENT</element>

CONTENT can be nothing, text, or more tags.

Basic Page Example

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
  </head>
  <body>
    <header>
      <h1>Heading 1</h1>
    </header>
    <div>
      <p>Paragraph Tag</p>
    </div>
  </body>
</html>
<!-- end page example -->
