Daily Challenge

  • Go to github.com and create a new repository named "final-project"
  • Clone your final-project repository using SSH
  • Add the following to your final-project folder
    • about.html
    • contact.html
    • index.html
    • css folder
      • style.css
    • img folder
  • Add, commit and push your work to Github.
